# Review Reply Lab Operations Manual

## What this product does

Privacy-first review response drafts that teach owners how to reply with empathy and local voice.

## Primary workflow

1. Complete the purpose-built domain tool section.
2. Attach evidence in the evidence locker.
3. Complete the v1 workbench: Review Response Lab.
4. Resolve readiness warnings.
5. Certify release gates only with real project evidence.
6. Export JSON, CSV, Markdown, and print packets for the project folder.

## Domain outputs

- Positive reply pack
- Critical review response
- Tone guide

## Privacy and retention

Data is local-first. Export intentionally, store securely, and delete browser state after handoff if working on a shared device.
